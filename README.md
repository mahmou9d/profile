🔗 **Live Demo:** [https://mahmou9d.github.io/profile](https://mahmou9d.github.io/profile)

---

## ✨ Features

- 🌌 **Immersive Dark Theme:** Sleek and premium dark UI designed with a futuristic terminal visualization aesthetic.
- ⚡ **Lightning Fast Performance:** Code-splitting and lazy-loaded components ensuring instant page load times and snappy navigation.
- 🎨 **Dynamic Micro-animations:** Powered by **Framer Motion** (Motion React) for custom-engineered page transitions, physics-based scroll triggers, and terminal cursor effects.
- 📱 **Fully Responsive Layout:** Flawless user experience across desktop, tablet, and mobile displays.
- 📧 **Interactive Contact Form:** Secure contact forms integrated seamlessly with **Formspree**.
- 🛠️ **Seamless Routing & Hash Navigation:** Built with **React Router v7** using smooth scroll-to-hash mechanics for frictionless internal section scrolling.

---

## 🛠️ Core Stack & Tools

- **Core Framework:** React 18 & TypeScript
- **Build Tool:** Vite 6
- **Styling & UI:** Tailwind CSS, custom vanilla CSS variables, and Shadcn UI components
- **Animations:** Framer Motion (Motion React)
- **Icons:** React Icons & Lucide React
- **Forms:** Formspree React SDK
- **Routing:** React Router DOM v7

## 💻 Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have **Node.js** and **npm** installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mahmou9d/profile.git
   cd profile
   ```

2. **Install project dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:5173` (or the port specified by Vite).

### Production Build

To compile the application into static files for production deployment:
```bash
npm run build
```
This output is saved to the production directory (e.g. `dist`).

### Deployment

Deploy directly to **GitHub Pages** using:
```bash
npm run deploy
```
